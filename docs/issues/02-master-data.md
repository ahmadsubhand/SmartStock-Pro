# Issue 2: Master Data Management & Product Gallery

## Description
Implementasi fitur CRUD untuk data referensi (Warehouse, Category, Supplier) dan Katalog Produk lengkap dengan galeri gambar.

## Goal
Memungkinkan pengguna untuk mengelola data dasar yang dibutuhkan untuk operasional inventaris.

## Tasks
- [ ] Implementasi CRUD Warehouse (Inertia + React).
- [ ] Implementasi CRUD Category (Inertia + React).
- [ ] Implementasi CRUD Supplier (Inertia + React).
- [ ] Implementasi CRUD Product dengan fitur:
    - [ ] Pencarian produk berdasarkan SKU/Nama.
    - [ ] Filtering berdasarkan Kategori dan Gudang.
    - [ ] Pagination yang dioptimasi.
- [ ] Integrasi sistem upload gambar untuk Galeri Produk (Storage link/S3).
- [ ] Gunakan Form Request untuk validasi data di sisi server.

## Tech Stack
- Laravel 13 (Controllers, Form Requests)
- Inertia.js v3
- React 19
- Tailwind CSS
